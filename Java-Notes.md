# Java Notes

## 1. Data Types
- int: Integer numbers
- float: Decimal numbers
- char: Single character
- boolean: true/false

## 2. Operators
- Arithmetic: +, -, *, /, %
- Comparison: ==, !=, >, <, >=, <=
- Logical: &&, ||, !

## 3. Control Statements
- if, else, else if
- switch
- for, while, do-while

## 4. Functions
- Syntax:
```java
returnType functionName(parameters) {
    // code
}
